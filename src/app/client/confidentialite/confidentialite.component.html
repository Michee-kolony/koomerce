Politique de confidentialité

Bienvenue sur notre application e-commerce. La protection de vos données personnelles est une priorité pour nous. Cette politique explique comment vos informations sont collectées, utilisées et protégées.

1. Collecte des données

Lors de votre utilisation de notre application, nous pouvons collecter certaines informations personnelles telles que : votre nom, votre numéro de téléphone et votre adresse.

2. Utilisation des données

Les données collectées sont utilisées uniquement dans le cadre du fonctionnement de la plateforme. Elles servent principalement de références pour les vendeurs afin de traiter correctement vos commandes, organiser les livraisons et assurer un bon suivi client.

3. Partage des données

Vos informations peuvent être partagées avec les vendeurs uniquement lorsque cela est nécessaire pour la gestion de vos commandes. Nous ne vendons pas vos données personnelles à des tiers.

4. Protection des données

Nous mettons en place des mesures de sécurité pour protéger vos informations contre tout accès non autorisé, toute modification ou toute divulgation.

5. Vos droits

Vous avez le droit de demander la modification ou la suppression de vos données personnelles en nous contactant via l’application.

6. Contact

Pour toute question concernant cette politique de confidentialité, veuillez nous contacter via le support de l’application.
